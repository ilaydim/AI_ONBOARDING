# Genel Şirket Süreçleri

## Sprint Yapısı
- 2 haftalık sprint döngüsü
- Pazartesi: Sprint Planning
- Cuma: Sprint Review + Retrospective
- Günlük standup: 09:30, Slack üzerinden async

## Kullanılan Araçlar
| Araç | Kullanım |
|------|----------|
| GitHub | Kaynak kodu + PR yönetimi |
| Jira | Sprint ve görev takibi |
| Confluence | Dokümantasyon |
| Slack | İletişim |
| Datadog | Monitoring |
| Figma | UI/UX tasarım |

## Onboarding Süreci
1. İlk gün: Sistem erişimleri + güvenlik eğitimi
2. İlk hafta: Takım tanıtımı + mimari genel bakış
3. İlk ay: İlk PR + buddy sistemi
4. 3. ay: Tam bağımsız çalışma
