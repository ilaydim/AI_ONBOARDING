# Backend Ekibi

TechNova Suite'in sunucu tarafı mimarisini geliştiren ve operasyonunu yürüten ekip.

## Ekip Yapısı
- 12 backend mühendisi (3 junior, 5 mid, 4 senior)
- 1 Staff Engineer (mimari kararlar)
- 1 Engineering Manager

## Teknoloji Stack
- **Dil**: Python 3.11+ (FastAPI), Go (performans kritik servisler)
- **Veritabanı**: PostgreSQL (primary), Redis (cache), Elasticsearch (arama)
- **Mesajlaşma**: Kafka
- **Container**: Docker + Kubernetes
- **CI/CD**: GitHub Actions + ArgoCD

## Sorumluluklar
- REST ve GraphQL API geliştirme
- Veri modelleme ve migration yönetimi
- Performans optimizasyonu
- Güvenlik ve compliance
