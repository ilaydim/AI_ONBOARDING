# Frontend Ekibi

TechNova Suite'in kullanıcı arayüzlerini ve kullanıcı deneyimini tasarlayan ekip.

## Teknoloji Stack
- React 18 + TypeScript
- Vite (build tool)
- TailwindCSS + shadcn/ui
- React Query (server state)
- Zustand (client state)
- Playwright (E2E test)

## Sorumluluklar
- Component kütüphanesi geliştirme ve bakımı
- Performans optimizasyonu (Core Web Vitals)
- Erişilebilirlik (WCAG 2.1 AA)
- Cross-browser uyumluluk
